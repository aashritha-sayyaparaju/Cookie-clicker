# Cookie-clicker

A cookie clicker clone but make it boba. Click the cup, earn pearls, spend them on upgrades, watch the numbers go up forever.

How to Play:
Click the boba to earn pearls, buy upgrades from the shop. Some upgrades give you more pearls per click, others earn pearls automatically over time! Check your stats on the right.

Upgrades:
extra straw: +1 per click, $10 starting cost
tapioca scoop: +1/sec, $50 starting cost
brown sugar drizzle: +3 per click, $200 starting cost
boba machine: +5/sec, $500 starting cost
boba empire: +40/sec, $5000 starting cost

costs scale by 1.15x each time you buy one.
